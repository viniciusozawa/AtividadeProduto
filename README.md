# 🛒 Atividade de Produto

Projeto desenvolvido para gerenciamento de produtos, permitindo cadastro, listagem, alteração e exclusão de itens.

---

## 📌 Observação Importante

⚠️ Para executar corretamente o projeto:

> Utilize a branch **`feature/backend`**, pois a branch `main` contém alterações de estilização que podem não estar alinhadas com a atividade.

---

## 🚀 Funcionalidades

- ✅ Cadastro de produtos  
- 📋 Listagem de produtos  
- ✏️ Alteração de dados  
- ❌ Exclusão de produtos  

---

## 🛠️ Tecnologias Utilizadas

- Java (Servlets/JSP)
- JDBC
- MySQL
- HTML + CSS
- JSTL
